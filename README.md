# UART-ECHO-IN-C
📡 UART Communication in C (Embedded Simulation)
📌 Overview

This project demonstrates a basic implementation of UART (Universal Asynchronous Receiver/Transmitter) communication using the C programming language. It simulates low-level register-based UART operations typically used in embedded systems.

The program includes functions to:

Read a character from UART
Write a character to UART
Understand hardware-level communication using registers
⚙️ Features
Direct register access using pointers
Polling-based UART communication
Separate functions for transmit and receive
Lightweight and easy-to-understand embedded-style code
🧠 Concepts Used
Embedded C Programming
Memory-Mapped I/O
Bitwise Operations
UART Communication Protocol
Polling Mechanism
